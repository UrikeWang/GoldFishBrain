# GoldfishBrain**

是一款針對使用者設定的目的地，進行地點偵測，等到使用者到達目的地時，會發送順利抵達目的地的訊息給親朋好友，
應用上的發想：
 - 設定目的地並發送通知給親朋好友，當安全抵達目的地時，則自動通知親朋好友
 - 設定目的地並發送通知給親朋好友，當使用者快抵達目的地時，則自動通知在目的地附近的親朋好友可以出發會合
